```text
Hello SUMO community,

I am sharing a lightweight academic audit skill/checklist for researchers using Eclipse SUMO and TraCI for traffic signal control experiments.

It focuses on:

- TLS phase and movement-green consistency
- route/config/additional-file consistency
- paired controller comparison
- batch-run reproducibility
- tripinfo/summary/edgeData reporting
- fixed-time, actuated, max-pressure, data-informed, and MPC-style controller audits

Repository: [GitHub link]

This is an independent academic resource and is not affiliated with or endorsed by the Eclipse SUMO project, the Eclipse Foundation, or DLR.

Feedback is welcome, especially on common reproducibility failures that should be added to the checklist.
```
