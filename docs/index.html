What it is

A Markdown-based Codex/Claude skill package that guides an agent through SUMO configuration, controller, output, and claim-boundary audits.

Who it is for

Researchers using Eclipse SUMO for fixed-time, actuated, max-pressure, NEMA, data-informed, or MPC-style traffic signal control.

What it prevents

Broken routes, unsafe TLS phases, unpaired comparisons, overwritten outputs, invalid metrics, and unreproducible batch runs.

Current scope

The current release focuses on SUMO/TraCI traffic signal control experiments. It is not yet a general-purpose audit skill for every Eclipse SUMO use case.

Future skills may add audit support for routing and demand, emissions and energy, public transport, pedestrian and intermodal scenarios, AV/CAV and co-simulation workflows, calibration, safety analysis, and simulation-mode comparisons.

simulation-helper-skill-for-eclipse-sumo

Use for planning, reviewing, comparing, and writing evidence-bounded claims from SUMO/TraCI signal-control experiments.

debugging-helper-skill-for-eclipse-sumo

Use for closed-loop diagnosis of route, TraCI, TLS, demand, detector, output, seed, completion, and reproducibility failures.

Design approach

The skills combine official SUMO documentation, SUMO community troubleshooting patterns, public traffic-simulation code lessons, and practical experiment-audit heuristics.

Audit checks

  • Network, route, config, detector, and additional-file consistency.
  • TLS phase, movement-green, yellow, all-red, and NEMA evidence checks.
  • Paired controller comparisons across seeds, demand, horizons, and outputs.
  • Completion-aware reporting for stopped, unfinished, teleported, or invalid runs.
  • Metric definitions for travel time, delay, stops, throughput, queue, emissions, and fairness.
  • Claim boundaries for baselines, ablations, sensitivity runs, and debugging fixes.

Limitations

This repository provides agent instructions, checklists, and audit procedures. It does not certify that a SUMO experiment is correct, publishable, or officially validated.
