# Ingolstadt Review HTML Cockpit

This folder contains a committed HTML human-review cockpit generated from the example Ingolstadt Torii visual-detail network.

Open `ingolstadt_workflow_review.html` to inspect gate status, review queue, network overview, problem overlay, reference comparison, and dense-junction cluster zooms.

The HTML is diagnostic evidence only. It does not prove the Torii network is equivalent to the TUM reference or safe for destructive junction aggregation.
